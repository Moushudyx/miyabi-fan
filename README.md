## 这是什么站点?

这是一个以星见雅为主题的同人站点, 包含角色信息、同人作品、视觉展示与官方情报等内容

## 我也想添加一些东西?

如果想要修改同人作品这一栏的内容, 请编辑 `src/data/fanWorks.ts` 文件, 按照已有的格式添加新的作品信息, 然后上传必要的资源文件到 `public/works/` 目录下

如果想要添加模型、模型动作文件, 请上传到 `public/model/miyabi/` 目录下, 并在 `src/data/mmdModels.ts` 中添加对应的模型信息

## 页面设计

见 `/design` 目录下的设计文档

## 使用技术

Vue 3 + TypeScript + Nuxt 4
